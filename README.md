## mrCrab mini project

_Mini mock bank system for java language refresher_

## Technologies used

## Java 21 LTS
Java 21 is the latest LTS release of the Java programming language.
It is a long-term support release and is recommended for production use.
It is fully backwards compatible with Java 8.

### BigDecimal
BigDecimal class is used for currency calculations.
It is useful for financial calculations and data handling as it
avoids floating point arithmetic errors.

### Records
Records are used to store data in a structured way.
They are immutable and thread-safe. Introduced in Java 14, they are
similar to classes but with less boilerplate.

## How to run
To run the project, clone the repository and run the main method in 
the Main class.
